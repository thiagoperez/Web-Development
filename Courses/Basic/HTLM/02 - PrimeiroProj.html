# Aula 02: Criando o Primeiro Projeto em HTML Nesta aula, aprendemos a criar o primeiro arquivo em HTML. Alguns pontos importantes foram destacados: - Ao criar um arquivo em HTML, é necessário prestar atenção na extensão correta do arquivo. O padrão para arquivos HTML é usar a extensão .html. - A extensão correta do arquivo é importante para que os navegadores e outras ferramentas reconheçam e interpretem corretamente o conteúdo HTML. É essencial seguir o padrão de nomenclatura e utilizar a extensão .html ao criar arquivos HTML para garantir a correta renderização e interpretação do código pelos navegadores.
